Введение в Python
####################

АННОТАЦИЯ

Электронное учебно-методическое пособие «Основы Python» предназначено для обучающихся по профессиям и специальностям направления подготовки 09.00.00 Информатика и вычислительная техника, в том числе при изучении дисциплины «Основы алгоритмизации и программирования».
Пособие включает теоретический материал, задания для самостоятельного выполнения, методические указания и примеры выполнения заданий. Учебный материал в пособии представлен по принципу «от простого к сложному»; задания, предлагаемые к выполнению, содержат иллюстрации, гиперссылки на источники и пошаговые практические рекомендации.

Для освоения основ языка программирования Python обучающемуся предлагается, используя удобную систему навигации, последовательно переходить от темы к теме, изучать теоретический материал, разбирать примеры и выполнять практические задания.

Электронное учебно-методическое пособие «Основы Python» может быть использовано преподавателем для подготовки и проведения аудиторных учебных 
занятий, для организации самостоятельной работы обучающихся, а также при реализации дополнительных общеразвивающих и профессиональных программ для различных категорий слушателей.
