SSH
======

SSH (англ. Secure SHell — «безопасная оболочка») – сетевой протокол прикладного уровня передачи данных, позволяющий производить безопасное и защищенное управление операционной системой и данными. Дает возможность шифрования передаваемых данных и паролей. Также позволяет передавать любой другой протокол. Эта служба была создана в качестве замены Telnet, не выполняющему шифрование и использует криптографические техники, чтобы обеспечить шифрование всего что передается между между сервером и пользователем. SSH предоставляет механизм для авторизации удаленного пользователя, передавая команды клиента хосту и возвращая ответ обратно клиенту.
